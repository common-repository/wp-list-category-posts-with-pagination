== Description ==
Wp List Category Posts With Pagination allows you to list posts from a category into a post or page with pagination using the [es-catlist] 
shortcode. This shortcode accepts a category id so attribute would be "cat=Category_ID", 
the order would be default according to post date, and the number of posts will dispaly according to pagination option. 
Usage: [es-catlist cat=Category_ID].
